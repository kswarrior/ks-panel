## ┃🔷 Installation
- 📦 Install Packages
```bash
bash <(wget -qO- https://raw.githubusercontent.com/kswarrior/ks-panel/main/package/kspanel.sh)
```
- 🗃️ Setup Database
``` bash
kspanel database
```
- 📥 Install Panel
```bash
kspanel install
```
- 🚀 Start Panel
```bash
kspanel start
```
